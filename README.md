# Fermium

Fermium is a fork of Celeritas for CleanroomMC / Minecraft 1.12.2.

It aims to provide a modern performance-focused rendering engine with experimental shader support, based on the Celeritas/Oculus rendering pipeline.

## Status

Fermium is currently experimental.

Shader support is incomplete. Some shaderpacks may not load, may render incorrectly, or may crash the game.

## Requirements

- Minecraft 1.12.2
- CleanroomMC
- Java 17+
- Forge/Cleanroom-compatible modpack

## Incompatibilities

Fermium is not compatible with OptiFine.

Mods that heavily modify the rendering pipeline may cause visual bugs or crashes.

## Shader Support

Fermium includes experimental shader support.

Current goal:

- Add a working Shaders button
- Load shaderpacks from the `shaderpacks` folder
- Support a small list of tested shaderpacks first
- Improve compatibility over time

Recommended shaderpacks will be listed once tested.
